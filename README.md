# Deepak's Portfolio Website

A modern, responsive portfolio website for a Data Analyst showcasing skills, projects, and professional information.

## Features

- **Modern UI/UX**: Beautiful, clean design with smooth animations
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling, hover effects, and animations
- **Easy to Customize**: Well-structured code for easy updates and modifications
- **Future-Ready**: Built with scalability and feature additions in mind

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Section**: Personal information and professional background
3. **Skills Section**: Grid layout showcasing technical skills
4. **Projects Section**: Featured projects with descriptions and tags
5. **Contact Section**: Contact form and social media links
6. **Footer**: Additional links and copyright information

## Getting Started

1. Open `index.html` in a web browser
2. No build process required - it's a static website
3. Customize content, colors, and sections as needed

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Content
- Update personal information in `index.html`
- Add/remove skills in the skills section
- Add your projects in the projects section
- Update contact information
